# WindowsFinder

**WindowsFinder** 是一款专为 Windows 系统信息快速采集与分析设计的本土化工具，旨在通过调用微软官方公开的 Windows API（NetUserEnum、WMI、注册表、事件日志等），高效、准确地发现和汇总 Windows 主机上的关键信息，极大地方便渗透测试、红队评估、蓝队自查以及系统运维审计工作。

当前功能一览
----------------
- 系统账户信息采集（支持 NetUserEnum 的 level 0~3、10、11、20、23，可按需调整信息详细程度）
- RDP 相关事件日志深度解析（事件 ID 21、25、1149 等，独立模块，解决超大日志漏采问题）
- 系统进程完整信息采集
- 计划任务枚举（开发中）
- 开机启动项枚举（开发中）
- 网络连接与监听端口信息采集（开发中）
- 未来计划：简易 GUI / Web 图形化界面（长期目标）
